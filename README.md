# Pills tracker web app
Repository for pills tracker web app built with [electron-boilerplate](https://github.com/szwacz/electron-boilerplate.git).

# Quick start

```
yarn install
npm start
```

# Testing

## Unit tests

```
npm test
```

## End to end tests

```
npm run e2e
```

## Code coverage

```
npm run coverage
```

# Making a release

To package the app into an installer (in dist) use command:

```
npm run release
```
